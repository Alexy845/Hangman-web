Hangman Web Rules

What's Hangman Web ?

The goal of the game is simple :

Guess all the letters that make up the secret magic word of Santa Claus and this with a limited number of attempts, but be careful because the hanged man is none other than Santa Claus himself !

How to play :

1. You start the game with some given letter(s) to help you find the word.

2. Each time the player guesses a letter, it is displayed. Otherwise, Santa Claus sees his condemnation a little closer and you lose an attempt.

3. When the player uses a letter, it is added to the list of letters you already have.

4. If you think you know the magic word in whole, you have the possibility of proposing it, but be careful because if you are wrong you will lose two lives and Santa Claus will approach the scaffold more quickly.

5. The game ends when you have found the word... or when you have no more life.
